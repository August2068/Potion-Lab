## Installation
The project is deployed online at: [here](url github pages)

If you'd like to run the project locally, you can either **download** or **clone** the project to your computer.

### Download project
You can download the ZIP file of the project by clicking the green **Code button** at the top right of this repository (on GitHub), then selecting **Download ZIP**.
Once downloaded, **unzip the file** and **open the `index.html` file in your web browser** to view the project.

### Clone project
To clone the repository using Git, open a terminal and run:
```
git clone *https://github.com/August2068/Potion-Lab.git*
```
Replace `your-username` with your actual GitHub username if necessary.
Once cloned, navigate to the project folder and open `index.html` in your preferred browser.
